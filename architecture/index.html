8 个抗体模式 + 10 步排障流程

抗体触发动作
timeout-retry网络超时指数退避重试
rate-limit-wait频率限制等待冷却后继续
model-switch模型不可用降级链切换
path-fix路径不存在自动创建/修正
format-heal格式异常重新解析/修复
state-reset状态不一致重置为已知良好状态
context-trim上下文超限压缩/裁剪上下文
escalate自愈失败升级到人类

完整的 10 步排障流程覆盖：识别 → 分类 → 匹配抗体 → 执行 → 验证 → 记录 → 升级判定 → 通知 → 回溯 → 归档。

三层资料结构。LLM 写/维护 wiki，人类提供 source

Source → Summary → Published 逐层精炼

层路径内容维护者
Source20-资料/原始文件（PDF/截图/笔记）人类
Summary10-项目/摘要结构化提取摘要LLM
PublishedPRD/设计方案经评审终稿LLM + 人类评审

设计原则：人类只管输入 Source；LLM 负责提取、整理、发布；人类保留 Review 否决权。
